Security Layers

IMPLEMENTATION
🔍
SAST — Static Analysis
Code Security
  • SonarQube scans for SQL injection, XSS, hardcoded credentials
  • OWASP dependency check on all third-party libraries
  • Checkmarx integrated into Jenkins pipeline pre-build
  • Fail pipeline if CRITICAL or HIGH severity found
🐳
Container Security
Docker + Registry
  • Trivy scans Docker image for OS & library CVEs
  • Non-root user inside container (USER 1001)
  • Read-only filesystem; minimal base image (distroless)
  • Image signing with cosign / Notary v2
☸️
K8s Security
Orchestration
  • RBAC with least-privilege service accounts per namespace
  • PodSecurityPolicy / OPA Gatekeeper admission control
  • Network Policies restrict pod-to-pod communication
  • Secrets managed via HashiCorp Vault, not K8s Secrets
🏥
HIPAA Compliance
Healthcare Regs
  • PHI data encrypted at rest (AES-256) and in transit (TLS 1.3)
  • Audit logs for all PHI access — immutable storage
  • Role-based access: doctor, nurse, admin, patient tiers
  • Automatic session timeout and MFA enforcement
📡
DAST & Runtime
Dynamic Testing
  • OWASP ZAP automated scan against staging environment
  • Burp Suite Enterprise for authenticated scan flows
  • Falco runtime security — detect container anomalies
  • Prometheus + Alertmanager for security metric alerts
🔐
Identity & Access
Zero Trust
  • OAuth 2.0 + OIDC with Keycloak identity provider
  • JWT tokens with short expiry and refresh rotation
  • API Gateway (Kong) with rate limiting and WAF rules
  • Zero-trust: verify every request regardless of origin
